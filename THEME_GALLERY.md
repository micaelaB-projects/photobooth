## 🔄 Mixing Themes

You can combine theme override files!

```html
<!-- Classic Photobooth colors with Game Boy typography -->
<link rel="stylesheet" href="css/style.css">
<link rel="stylesheet" href="css/theme-gameboy.css">
```

Or combine effects:

```html
<!-- Synthwave colors with Classic saturation -->
<link rel="stylesheet" href="css/style.css">
<link rel="stylesheet" href="css/theme-synthwave.css">
<link rel="stylesheet" href="css/theme-disco-fever.css">
<!-- Last one wins on conflicts -->
```

## 🎨 Creating Theme Variations

Each theme file contains CSS overrides. Stack them for unique combinations:

```html
<!-- Custom Mix: Disco + Synthwave hybrid -->
<link rel="stylesheet" href="css/style.css">
<link rel="stylesheet" href="css/theme-disco-fever.css">
<!-- Override with some Synthwave elements -->
<style>
    .photobooth {
        box-shadow: 0 0 60px rgba(233, 69, 96, 0.6),
                    0 0 40px rgba(255, 215, 0, 0.4);
    }
</style>
```
