## 🚀 Getting Started Steps

### Step 1: Verify Setup
```
✓ Apache running
✓ MySQL running
✓ PHP 7.0+
```

### Step 2: Configure Database
Edit `db_config.php` with your credentials

### Step 3: Initialize
Open `http://localhost/photobooth/db_setup.php`

### Step 4: Launch
Open `http://localhost/photobooth/`

### Step 5: Choose Theme
Edit `index.php` and add theme CSS link

### Step 6: Start Capturing!
- Click Start Camera
- Grant permissions
- Click Capture
- Download photos
